La marca pre       preserva        espacios    en      blanco
y saltos de linea

La marca pre también es buena para mostrar codigo de programación:

for i = 1 to 10
    print i
next i

También se puede usar la marca code

for i = 1 to 10 print i next i

pero code no respeta espaciado ni salto de linea
